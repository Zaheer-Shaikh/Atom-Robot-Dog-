# Atom-Robot-Dog-
The Atom which is a robot dog made for my COE project from the year 2025-26. During my 3rd year B-tech. Designing, printing, connection and assembly took us about 5 months. The project cost us about $200 and was funded by my Uni.

Our project faced alot of Power supply issues, we basically destroyed about 6-7 Esp32s due to overheating.
Even while using 2 buck converters constantly running the robot causes alot ofv overheating.
our battary was 2400mAh and 3.8v if the voltage drops below 3.6v the robot is not able to take the load and falls.
